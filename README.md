# Organizér na stojací stůl
Model je dostupný na [Printables](https://www.printables.com/model/1756178-headphone-stand-for-standing-desk)

3D tištěný organizér určený k upevnění na stranu nohy stojacího stolu.
Cílem projektu je vytvořit praktický a kompaktní držák na věci, které je potřeba mít u sebe, ale nemají překážet na pracovní ploše ani v prostoru pro nohy.

Organizér bude obsahovat držák na sluchátka přes hlavu, vedení kabelu, výsuvný věšák a malý prostor pro detaily.
